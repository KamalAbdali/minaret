About Minaret

This program computes Islamic prayer schedules, the direction of the Qibla, and various data related to the Hijri calendar. It is free for personal use.

You may distribute copies of this program to others at no charge. Please give only unmodified copies to others, and include this documentation with the program.

The most recent version of this program is available for download from http://geomete.com/minaret/minar40.zip.
