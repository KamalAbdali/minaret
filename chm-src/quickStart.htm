Quick Start

Installing Minaret
  1. Create a new directory, for example, C:\Minaret.
  2. Copy the downloaded file Minaret.zip into that directory.
  3. Unzip Minaret.zip using a utility. In Windows, double-clicking on the zip file usually does this automatically. You can also do it using a utility such as 7zip or Winzip.
All the files needed to run Minaret are in the Minaret directory. The *.DLL and *.OCX files may be moved to the directory Windows\System. But doing this may overwrite other versions of these files, and affect proper execution of other applications. It is best to simply leave these files in the Minaret directory.

Running Minaret

 Use the Run... menu from Window's Start, or double-click on Minaret.exe in Windows Explorer.

Setting Current Location

 When Minaret starts, go to the Location Menu, and select Open Gazetteer from it. A large window will open. The box at top left shows the Current Location. This is the city for which Minaret does all computations, such as calculating prayer hours or the qibla direction, converting calendar dates, etc. From the scroll list below that box, click on your city. (If your city is not in the Gazetteer, then you have to enter your city’s name and geographical information by first pressing the Create New Entry button.) The name and geographical data will appear in the boxes at right. Verify the information, then press the Make it Current Location button. Then press the Close button to close the gazetteer.

Setting Options

 There are some jurisprudential variations in determining the prayer times for fajr, asr, and isha. Go to the Options menu, and select the methods according to your preferences.

Getting Results

 Now you are ready to compute with Minaret. Use the Prayer menu to compute prayer hours for any day, month, or year. Use the Qibla menu for determining the qibla direction. Use the Calendar menu to convert dates between the Hijri and Gregorian calendars or to get New Moon information helpful in determining Hijri dates.
